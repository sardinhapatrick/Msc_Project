Comprendre PATL

Un langage Déclaratif avec une mécanique de programmation par block

Le langage PATL se base sur React, une bibliothèque Javascript donnant la possibilité de facilement créer des interfaces utilisateurs interactives.
Les notions principales de React réutilisées pour notre langage sont le système de composants, de propriétés, d'état global ainsi que de fonctions de rendu et de mise à jour.
Le système de composants nous permet de dissocier le code et de considérer chaque morceau de manière indépendante dans le but de faciliter la programmation d'applications graphiques.

Utilisation d'abstractions

PATL se base sur Rendery, un moteur de rendu 2D/3D moderne qui est écrit en Swift et basé sur OpenGL dans le but de réutiliser ses abstractions pour la programmation graphique. Ces dernières vont de la création d'une application à la manipulation d'objets 3D dans une scène.

Utiliser ses propres types abstraits

Le langage PATL est un DSL prenant base sur le langage Swift, un langage de programmation haut niveau, simple, performant et sûr.
Le système de structures de ce langage permet ainsi aux programmeurs de mettre en place aisément leurs propres types abstraits et donc de développer leur application graphique comme bon leur semble.
