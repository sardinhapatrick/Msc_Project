Mise en place d'un Projet PATL

Swift

PATL est un DSL (Domain-Specific Langage) basé sur le langage Swift.

La version Swift 5.3 ou plus est nécessaire et peut être téléchargée sur le site officiel.

PATL standard library

L'import de la librairie standard de PATL est essentielle pour le fonctionnement d'une application PATL. Cette librairie peut être obtenue en clônant le dépôt Github avec la commande git clone https://github.com/sardinhapatrick/Msc_Project.git.

Swift Package Manager

Pour configurer le projet PATL, il est nécessaire de modifier le Package.swift fourni. La librairie PATL possède une dépendance à Rendery (branche "impRendery").

"project_name" représente le nom de votre projet. 
										
// swift-tools-version:5.3
// The swift-tools-version declares the minimum version of Swift required to build this package.
import PackageDescription

let package = Package(
    name: "project_name",
    products: [
      .library(name: "PATL", targets: ["PATL"]),
      .executable(name: "project_name", targets: ["Msc_Project"]),
    ],
    dependencies: [
        // Dependencies declare other packages that this package depends on.
        // .package(url: /* package url */, from: "1.0.0"),
        .package(url: "https://github.com/sardinhapatrick/Rendery.git", .branch("impRendery")),
    ],
    targets: [
        // Targets are the basic building blocks of a package. A target can define a module or a test suite.
        // Targets can depend on other targets in this package, and on products in packages this package depends on.
        .target(name: "PATL", dependencies: ["Rendery"]),
        .target(name: "projet_name", dependencies: ["PATL"]),

        .testTarget(name: "project_nameTests", dependencies: ["project_name"]),
    ]
)
